# 步骤 03：新增子目录扫描

## 目标

扫描主题根目录下的 `结果微调/XX/` 和 `实施步骤/XX/` 子目录，将它们作为结果微调单元和子主题单元纳入 `summary_input.json`。

## 涉及文件

- `app/powershell/summary/PrepareSummaryInput.ps1`

## 具体操作

### 1. 在 `PrepareSummaryInput.ps1` 中新增函数 `Get-SubUnits`

在 `Add-ImplementationRound` 函数之后插入：

```powershell
function Get-SubUnits {
    param(
        [string]$Dir,
        [string]$SubDirName,
        [array]$LogEntries
    )
    $subUnits = @()
    $subRoot = Join-Path $Dir $SubDirName
    if (-not (Test-Path $subRoot -PathType Container)) {
        return $subUnits
    }

    $childDirs = Get-ChildItem -LiteralPath $subRoot -Directory -Force | Sort-Object Name
    foreach ($child in $childDirs) {
        $requirementFile = Join-Path $child.FullName '需求.txt'
        if (-not (Test-Path $requirementFile)) {
            continue
        }

        $childRounds = Get-ThemeRounds -Dir $child.FullName -LogEntries $LogEntries
        $childRounds = Add-ImplementationRound -Dir $child.FullName -Rounds $childRounds -LogEntries $LogEntries

        if ($childRounds.Count -eq 0) {
            continue
        }

        $subUnits += [PSCustomObject]@{
            name            = $child.Name
            path            = $child.FullName
            rounds          = [PSCustomObject]@{
                total  = $childRounds.Count
                items  = $childRounds
                phases = @()
            }
            totalTime       = $null
            totalHumanChars = ($childRounds | Measure-Object -Property humanChars -Sum).Sum
            totalAiChars    = ($childRounds | Measure-Object -Property aiChars -Sum).Sum
        }
    }

    return $subUnits
}
```

### 2. 在主流程中调用

在生成主轮次之后调用：

```powershell
$resultFineTunings = Get-SubUnits -Dir $themePath -SubDirName '结果微调' -LogEntries $logEntries
$subThemes = Get-SubUnits -Dir $themePath -SubDirName '实施步骤' -LogEntries $logEntries
```

### 3. 更新输出结构

将 `$output` 中的 `rounds` 从数组改为对象结构，并增加 `resultFineTunings` 和 `subThemes`：

```powershell
$output = [PSCustomObject]@{
    themeName         = Split-Path -Leaf $themePath
    themePath         = $themePath
    generatedAt       = (Get-Date -Format 'yyyy-MM-dd HH:mm:ss')
    files             = [PSCustomObject]@{ ... }
    rounds            = [PSCustomObject]@{
        total  = $rounds.Count
        items  = $rounds
        phases = @()
    }
    resultFineTunings = $resultFineTunings
    subThemes         = $subThemes
    stageTime         = $stageTime
    totalTime         = $totalTime
    totalHumanChars   = $totalHumanChars
    totalAiChars      = $totalAiChars
}
```

注意：此时 `totalTime` / `totalHumanChars` / `totalAiChars` 仍按原逻辑计算，在下一步骤中再改为全量合并。

## 验证

在「周报时间」主题上运行 `PrepareSummaryInput.ps1`，检查：

- `resultFineTunings` 数组非空，每个元素包含 `name`、`path`、`rounds`。
- `subThemes` 数组非空（如果存在 `实施步骤/XX/`）。
- 每个子单元的 `rounds.items` 若含 `实施文档.md`，最后一轮为 `category = "实施确认"`。
