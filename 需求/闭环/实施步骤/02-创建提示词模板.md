# 02：创建提示词模板

## 目标

创建 `app/templates/open_md_prompt.txt`，作为"打开MD"开关开启时追加到提示词末尾的模板。

## 创建文件

`app/templates/open_md_prompt.txt`

## 文件内容

```text
补充指令：如果你在本轮回复中生成了 Markdown 文件，请在文件保存完成后，执行以下命令用 IDEA 打开它。注意：不要修改脚本路径，只需把 `<Markdown文件绝对路径>` 替换为你实际生成的 Markdown 文件绝对路径。

powershell -ExecutionPolicy Bypass -File "{{scriptPath}}" -FilePath "<Markdown文件绝对路径>"
```

## 占位符说明

- `{{scriptPath}}`：由 AIProcess 在运行时替换为 `app/OpenMarkdown.ps1` 的绝对路径。
- `<Markdown文件绝对路径>`：保留给模型替换。模型知道自己实际生成了什么文件，因此由它填入真实路径。

## 设计要点

- **脚本路径预填**：AIProcess 替换 `{{scriptPath}}`，模型不需要自己找脚本位置。
- **Markdown 路径由模型提供**：AIProcess 不预填具体 Markdown 文件路径，避免猜错。
- **命令格式固定**：模型只需要替换文件路径，其余部分原样复制执行。

## 验收标准

- 文件 `app/templates/open_md_prompt.txt` 存在且内容正确。
- `{{scriptPath}}` 占位符存在且唯一。
- `<Markdown文件绝对路径>` 占位符存在且唯一。
