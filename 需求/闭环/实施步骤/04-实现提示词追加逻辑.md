# 04：实现提示词追加逻辑

## 目标

在 `app/AIProcess.ahk` 中新增 `AppendOpenMdPromptIfNeeded` 函数，并在 `复需求`、`复回复`、`复关系` 三个入口调用它。

## 修改文件

`app/AIProcess.ahk`

## 具体修改

### 1. 新增追加函数

在 `AppendNoModifyPromptIfNeeded` 或 `AppendReplyImplementationTailIfNeeded` 附近新增：

```ahk
AppendOpenMdPromptIfNeeded(content) {
    global AppConfig, TemplateDir
    if (!AppConfig["OpenMdWithIdea"]) {
        return content
    }

    templatePath := TemplateDir "\open_md_prompt.txt"
    if (!FileExist(templatePath)) {
        return content
    }

    template := FileRead(templatePath, "UTF-8")
    template := StrReplace(template, "{{scriptPath}}", AppConfig["OpenMdScriptPath"])

    baseContent := RTrim(content, "`r`n")
    return baseContent "`r`n`r`n" template
}
```

### 2. 修改"复需求"入口

找到 `CopyRequirementPrompt()` 函数，在现有逻辑后添加调用：

```ahk
content := LoadTemplate("requirement_prompt.txt")
content := StrReplace(content, "{{filePath}}", CurrentDir "\需求.txt")
content := AppendNoModifyPromptIfNeeded(content)
content := AppendOpenMdPromptIfNeeded(content)
A_Clipboard := content
```

### 3. 修改"复回复"入口

找到 `CopyReplyPrompt()` 函数，在现有逻辑后添加调用：

```ahk
content := LoadTemplate("reply_prompt.txt")
content := StrReplace(content, "{{filePath}}", currentReplyFile)
content := StrReplace(content, "{{nextVersionFile}}", nextVersionFile)
content := AppendReplyImplementationTailIfNeeded(content)
content := AppendNoModifyPromptIfNeeded(content)
content := AppendOpenMdPromptIfNeeded(content)
A_Clipboard := content
```

### 4. 修改"复关系"入口

找到 `CopyContextRelations()` 函数，在现有逻辑后添加调用：

```ahk
content := BuildContextRelationsText()
content := AppendNoModifyPromptIfNeeded(content)
content := AppendOpenMdPromptIfNeeded(content)
A_Clipboard := content
```

## 设计要点

- **开关控制**：仅在 `AppConfig["OpenMdWithIdea"]` 为 true 时追加。
- **模板不存在则跳过**：避免模板缺失导致功能崩溃。
- **脚本路径预填**：`{{scriptPath}}` 替换为 `app/OpenMarkdown.ps1` 的绝对路径。
- **Markdown 路径不预填**：保留 `<Markdown文件绝对路径>` 占位符，由模型自行替换。
- **追加位置**：在 `AppendNoModifyPromptIfNeeded` 之后追加，确保补充指令出现在所有约束之后。

## 验收标准

- 开启"MD开"时，点击"复需求"、"复回复"、"复关系"，复制的提示词末尾都包含 PowerShell 打开命令。
- 提示词中的脚本路径为 `app/OpenMarkdown.ps1` 的绝对路径。
- Markdown 文件路径位置保留为 `<Markdown文件绝对路径>` 占位符，未被 AIProcess 替换为具体路径。
- 关闭"MD开"时，三个入口复制的提示词末尾都不包含打开命令。
